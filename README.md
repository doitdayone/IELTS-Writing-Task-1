# IELTS-Writing-Task-1

📒 Sổ tay từ vựng mô tả biểu đồ cho IELTS Writing Task 1 — web tĩnh HTML/CSS/JS, mở offline được.

## Tính năng

- **Sổ tay** ([index.html](index.html)): 6 phần từ vựng (Overview & Introduction, Key Expressions, So sánh, Body & mốc thời gian, Trends, Linking) — **93 từ/cụm từ** kèm nghĩa tiếng Việt.
- Bấm vào từ → popup **10 câu ví dụ** tự nhiên theo văn phong Task 1, từ khóa được highlight.
- Bấm 🔊 trên từ hoặc từng câu để **nghe phát âm** (Web Speech API — nên dùng Edge/Chrome).
- **Flashcards** ([flashcards.html](flashcards.html)): tick chọn phần muốn ôn, lật thẻ 3D, chấm ✅ Đã thuộc / ❌ Chưa thuộc, ôn lại riêng các từ chưa thuộc. Phím tắt: `Space` lật, `←/→` chấm, `S` nghe.
- Giao diện giấy kẻ ô + sticky note pastel, font viết tay Patrick Hand (hỗ trợ tiếng Việt, nhúng offline), responsive cho mobile.

## Cách dùng

Mở `index.html` bằng trình duyệt là xong — không cần cài đặt gì.
